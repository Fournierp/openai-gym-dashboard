# openai-gym-dashboard

## Introduction

OpenAI Gym Dashboard is a web app designed with Plotly's Dashboard Tool. This lightweight interface is a great tool to visualize the training of your Reinforcement Learning or Artificial Intelligent Agent. 

## Installation

Firstly, ensure that you have pip install. In which case follow these steps using the command line:

```
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
python get-pip.py
```

Then install the required libraries listed in the requirements.txt
```
pip install -r requirements.txt
```

## Usage Example
```
python index.py
```
